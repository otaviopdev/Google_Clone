## 💻 Sobre:

Projeto que utilizei como treinamento, utilizando HTML5 e CSS3 de forma responsiva.

## 📝 Características:

Se abrir a aplicação no celular, tablet ou qualquer tipo de tela, a aplicação se adapta.

## 🏁 Instalação:

Antes de começar, é bom ter um editor para trabalhar com o código, como: [VSCode](https://code.visualstudio.com/). Clone este repositório usando git clone: https://github.com/otaviopdev/Google_Clone.git ou clique em Code/Download ZIP. Clique no arquivo "index.html" e veja a aplicação.

## 🛠️ Tecnologias Utilizadas:

Principais tecnologias usadas neste projeto:

- [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML)
- [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS)
